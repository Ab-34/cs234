# CS234 Project: Bridging Offline-Online Learning in Robot Manipulation with Intelligent Reward Shaping

This work studies how to bridge offline pre-training and online refinement in robotic manipulation on the robomimic Lift task. It evaluates Behavior Cloning with PPO fine-tuning, heuristic reward shaping, and Vision-Language Model (VLM) rewards, identifying a persistent “grasp-lifting gap” in simpler reward designs. A curriculum-based reward that decomposes the task into sub-skills achieves the best performance (93.4% success), outperforming both heuristic and VLM-based approaches. Results highlight the importance of structured reward design for handling contact-rich manipulation.

## Scripts

### `src/ppo.py` PPO Training
Trains a PPO agent (via Stable-Baselines3) on the Lift environment from scratch.

```bash
python src/ppo.py \
  --policy MlpPolicy \
  --timestep 500000 \
  --save_path ./ppo_trained/
```

### `src/bc_ppo.py` BC Pretraining + PPO Fine-tuning
1. Loads expert demonstrations from an HDF5 robomimic dataset.
2. Pretrains SB3's `ActorCriticPolicy` with behavior cloning (MSE loss on actions).
3. Initializes PPO with the BC weights and fine-tunes with RL.

```bash
python src/bc_ppo.py \
  --dataset ./low_dim_v15.hdf5 \
  --timestep 500000 \
  --save_path ./ppo_trained/bc_ppo/
```

### `src/evaluate.py` Evaluate PPO Policy
Loads a saved PPO model and runs evaluation episodes.

```bash
python src/evaluate.py \
  --model_path ./ppo_trained/ppo_model.zip \
  --episodes 20 \
  --full_loss
```

### `src/bc_evaluate.py` Evaluate BC Policy
Evaluates a behavior-cloned policy checkpoint.

```bash
python src/bc_evaluate.py \
  --model_path ./bc_policy.pth \
  --episodes 20
```

### `src/train_cnn.py` Train CNN Reward Model
Fine-tunes an [R3M](https://github.com/facebookresearch/r3m) (ResNet-50) backbone with a regression head to predict task-progress scores (-100 – 100) from frames, using `frame_scores.json` as supervision.

```bash
pip install r3m
python src/train_cnn.py
```

Expects `frame_scores.json` in the working directory (generated by `extract_scores.py`). Saves the best checkpoint to `checkpoints/r3m_reward_best.pt`.

### `src/extract_scores.py` Frame Scoring via Claude
Reads videos from a `playback/` folder, samples frames, and scores each frame (−100 to 100) using **Claude** via **Amazon Bedrock**. Saves frames to `img/` and writes `frame_scores.json`.

```bash
python src/extract_scores.py
```

Requires AWS credentials with Bedrock access (`us-east-1`). Model used: `claude-sonnet-4-6`.

### `src/visualize.py` — Visualization Utilities
Provides helper functions for plotting training curves (loss, velocity norms, etc.) and saving evaluation rollouts as videos.
